## üõ†Ô∏è 1. Setup & Installation

### Copy the environment template and configure your `.env`

```bash
cp .env.example .env
```

Update the `DATABASE` in your `.env` file based on your setup:

```yaml
DATABASE_CONNECTION=postgres
DATABASE_HOST=host.docker.internal
DATABASE_NAME=
DATABASE_USER=
DATABASE_PASSWORD=
DATABASE_PORT=
```

> üí° **Tip:** Use `host.docker.internal` when your PostgreSQL runs on the Docker host machine

## üê≥ 2. Docker Development

### Run application with Docker in seconds:

```bash
# Build the application image
make build

# Start the containers
make run

# View application logs
make logs
# See all available commands
make help
```
